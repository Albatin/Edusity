# Edusity

**Edusity** is a simple educational website built with React. It showcases course listings and a clean, responsive UI — ideal for learning or personal projects.

## 🚀 Features

- Built with React
- Responsive layout
- Course listing pages
- Basic navigation

## 🛠️ Tech Stack

- React
- HTML/CSS
- JavaScript
